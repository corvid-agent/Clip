# Clip

Clipboard history manager for macOS menu bar. Monitors the system clipboard and stores up to 50 items with pin support for quick access.

![Clip Screenshot](screenshot.png)

## Features

- Monitors NSPasteboard for new clipboard entries
- Stores up to 50 items with automatic rotation
- Pin important items to keep them permanently
- Click any item to copy it back to clipboard
- Monospaced dark theme UI

## Build

```
swift build
```

## Run

```
swift run Clip
```
